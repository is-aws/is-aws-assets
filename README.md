# Is This Hosted On AWS Assets

CSS, HTML, image and script assets for "Is This Hosted On AWS" HTML API.

Requirements:

  * [Node.js](http://nodejs.org/).

To set-up your environment execute:

    $ npm install

To build execute:

    $ npm run build

To build and watch:

    $ npm run watch

To view:

    $ open dist/index.html

To clean build:

    $ npm run clean

To deploy:

    $ npm run deploy
